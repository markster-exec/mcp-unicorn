# Existing MCP Apps & Servers — Sales and Finance Landscape

## MCP Ecosystem Overview

Thousands of public MCP servers exist with growing directories and registries. Business tools (sales/CRM/finance) are well-represented but often fragmented/siloed.

## Directories & Marketplaces

- **mcpservers.org** — Community-curated hub with categories and search.
- **mcpmarket.com** — ~21k+ servers listed; leaderboards, categories (features Apollo.io prominently).
- **Glama.ai** — Marketplace with visual previews.
- **Official MCP Registry** (registry.modelcontextprotocol.info)
- GitHub "awesome-mcp-servers" repos (PipedreamHQ, wong2, TensorBlock).
- Claude Connectors directory (75+ official), emerging OpenAI App Directory.

## Sales/CRM MCP Servers

**Moderately crowded** — dozens of wrappers for major platforms + sales-intelligence specials.

- **Apollo.io MCP** (multiple implementations) — Prospect/company search, contact/company enrichment, lead gen, CRM sync, outreach. Directly integrates with Claude/Cursor.
- **HubSpot official MCP** — Full CRM access (contacts, companies, deals, pipelines).
- **Salesforce MCP** (official + via Merge/Unified.to) — Leads, opportunities, accounts.
- **Microsoft Dynamics 365 Sales MCP** — Pipeline, deals, sales workflows.
- Others: Gong, Outreach/Klenty, SalesBlink/Salespype/Autobound, Odoo/Zoho CRM, Draup, unified "Sales Intelligence MCP" (LobeHub).
- **Merge.dev / Unified.to** CRM MCP layers — Abstract across CRMs.

## Finance/Accounting MCP Servers

**Moderately populated** — major accounting platforms + market/financial data.

- **Xero MCP** (official/Coupler.io) — Full accounting data, natural-language AI analyst.
- QuickBooks/Intuit MCP, Norman Finance MCP, Pennylane, Expensify, Zoho Books, FreshBooks, Wave.
- **Financial data**: Yahoo Finance MCP, Financial Modeling Prep (FMP), Alpha Vantage, EODHD, Polygon.
- **Puzzle.io** — MCP support via Zapier MCP server. Not a dominant standalone.

## Crowded vs Empty Categories

- **Crowded**: Developer tools, browser/OS automation, popular SaaS wrappers, web scraping.
- **Populated but fragmented**: Sales/CRM (many platform-specific; unified intelligence rarer), Finance/Accounting (majors covered; deep cross-system modeling rarer).
- **Relatively empty/underserved**: **Hyper-unified sales + finance** (e.g., real-time pipeline-to-P&L forecasting with enrichment), advanced predictive MCP Apps, monetized agent marketplaces.

## How Our Concept Differentiates

**Nothing dominates as a comprehensive, AI-native, end-to-end sales-finance intelligence platform** that tightly integrates:

- Company/deal enrichment + pipeline management
- Real-time financial modeling/forecasting tied to deals
- Deal analysis with cross-system insights (CRM + accounting + market data)
- Rich interactive MCP App dashboards

Most existing servers are siloed (one CRM or one accounting system) or basic data pipes. Our concept builds a higher-level, differentiated layer on top of the ecosystem.
