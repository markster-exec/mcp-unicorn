# AI Sales Companions in Chat — Competitive Landscape

## Key Finding

**No major startup is exactly replicating "end-to-end AI sales companion living natively in Claude/ChatGPT" as its core product (yet).** The closest activity is in the emerging MCP ecosystem.

## Startups Building In-Chat / MCP-Enabled Sales Companions

| Company                  | What                                                                                                        | Interface                           | Notes                                          |
| ------------------------ | ----------------------------------------------------------------------------------------------------------- | ----------------------------------- | ---------------------------------------------- |
| **Vendis.ai** (Greylock) | "Your intelligent AI-native sales companion." Conversational CRM, auto-updates pipeline from conversations. | Own chat/voice + MCP/REST APIs      | Early/private beta. Closest competitor.        |
| **Cliccc.ai**            | AI sales companion syncing IRL interactions to CRM.                                                         | Mobile-first app (Claude ecosystem) | Won Anthropic/Lovable hackathon at Slush 2025. |
| **Prismus.ai**           | "Agentic AI Sales Companion" for deep tech/scientific sales.                                                | Separate dashboard/platform         | Microsoft AI Accelerator 2026. No MCP mention. |
| **People.ai**            | MCP integration (Feb 18, 2026) for SalesAI. Query full revenue data in Claude/ChatGPT.                      | Native in Claude/ChatGPT via MCP    | Major player. "Stay in your workflow."         |
| **HeyReach**             | MCP server turns Claude into sales assistant for campaigns, personalization, reply analysis.                | Native Claude chat                  | Outbound/LinkedIn focused.                     |

## AI SDR (Replaces) vs AI Companion (Augments)

| Type                               | Examples                                                 | Model                                | Best For                   |
| ---------------------------------- | -------------------------------------------------------- | ------------------------------------ | -------------------------- |
| **AI SDR/BDR** (replacement)       | 11x (Alice), Artisan (Ava), MagicBlocks                  | Autonomous, operates independently   | High-volume prospecting    |
| **AI Companion/Copilot** (augment) | Vendis, Amplemarket Duo, Glyphic, SiftHub, People.ai MCP | Human-first, interactive in workflow | Higher win rates per human |

Our positioning: **companion/augment** — end-to-end but human-augmented in native chat.

## Who's Using MCP for Sales?

Exploding in 2025-2026:

- Microsoft Dynamics 365 Sales MCP (leads, emails, quotes in Claude/ChatGPT)
- HubSpot MCP (pipeline tasks, insights)
- People.ai (full revenue intel)
- HeyReach (Claude as sales assistant)
- Amazon Ads, Klaviyo, Granola, The Swarm, Float, Clarify
- User-built: "Claude MCP Agent closed $6K deal"

## VC Theses

- **Greylock**: Highlights Vendis. Signals interest in conversational/AI-native CRM companions.
- **a16z**: Agents/companions shifting from chatbots to proactive tools.
- No single "definitive VC thesis" on narrow "in-chat sales companion" category (still nascent post-MCP).

## How to Position

"The end-to-end AI sales companion that lives where you already work — in Claude or ChatGPT. Powered by MCP for real-time CRM/data access and actions."

- Vs. autonomous SDRs: "Not an AI employee that works alone — a true teammate that supercharges YOU."
- Vs. app-based copilots: "No new dashboard or Slack bot. Seamless in your daily LLM chat."
- Category: **Chat-native / LLM-resident / MCP-enabled Sales Companion.**
