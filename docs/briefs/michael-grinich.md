## Background

- **Nylas** — Co-founded. Email/calendar/contacts API platform. Successfully exited.
- **WorkOS** — Current company. Founded 2019. Enterprise identity and auth platform for B2B SaaS.
- **$80M Series B** — Major funding round
- **$30M+ ARR** — Significant revenue traction
- ~120 employees, San Francisco

## WorkOS Products

7 core products:

1. **AuthKit** — Drop-in authentication UI
2. **Single Sign-On (SSO)** — Enterprise SAML/OIDC
3. **Directory Sync (SCIM)** — User provisioning
4. **Fine-Grained Authorization (FGA)** — Permissions/RBAC
5. **Admin Portal** — Self-service for IT admins
6. **Security (Audit Logs)** — Compliance and audit trails
7. **Pipes** — Data integration layer
